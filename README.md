# kelfy87site

Персональная GitHub-витрина в темной черно-красной эстетике: неоновая аура, анимированный фон, 3D-карточка профиля, музыка и живые данные из GitHub API.

Проект сделан как легкая статическая страница без сборщиков и зависимостей. Достаточно открыть `index.html` в браузере или опубликовать папку на GitHub Pages.

## Что внутри

- **Живой профиль GitHub**: страница загружает аватар, био, статистику и репозитории пользователя `kelfy87`.
- **Топ репозиториев**: публичные репозитории сортируются по звездам и выводятся отдельными карточками.
- **3D-tilt карточка**: аватар реагирует на движение мыши и создает эффект глубины.
- **Canvas-фон**: мягкое звездное поле с параллаксом и красным свечением.
- **Лоадер**: стартовый экран с прогресс-баром и атмосферной подачей.
- **Музыкальная кнопка**: локальный `music.mp3` включается по клику, чтобы браузер не блокировал автозапуск.
- **Адаптивная верстка**: страница подстраивается под мобильные и десктопные экраны.

## Структура проекта

```text
kelfy87site/
├── index.html   # Вся разметка, стили и JavaScript сайта
├── music.mp3    # Локальный аудиотрек для музыкального переключателя

```

## Быстрый запуск

Самый простой способ:

```text
Открой index.html в браузере
```

Если браузер ограничивает загрузку аудио или запросы при открытии через файл, запусти локальный сервер:

```powershell
python -m http.server 8000
```

После этого открой:

```text
http://localhost:8000
```

## Публикация на GitHub Pages

1. Загрузи файлы проекта в репозиторий.
2. Открой настройки репозитория.
3. Перейди в **Pages**.
4. Выбери ветку и папку, где лежит `index.html`.
5. Сохрани настройки и дождись публикации.

Так как сайт полностью статический, отдельный backend или сборка не нужны.

## Настройка

Основная конфигурация находится в начале JavaScript-блока внутри `index.html`:

```js
const GH_USER = "kelfy87";
```

Чтобы сделать страницу для другого GitHub-профиля, измени значение `GH_USER`.

Также можно заменить локальные ассеты:

- `skin.png` - резервная картинка профиля, если GitHub API недоступен.
- `music.mp3` - трек, который включается кнопкой в правом нижнем углу.

Важно: имя `music.mp3` чувствительно к регистру на GitHub Pages. Если файл называется иначе, обнови путь в коде.

## Как работает GitHub API

Сайт обращается к публичным endpoint'ам GitHub:

```text
https://api.github.com/users/kelfy87
https://api.github.com/users/kelfy87/repos?per_page=100&sort=updated
```

Данные используются только на клиенте:

- логин;
- аватар;
- био;
- количество публичных репозиториев;
- followers / following;
- локация, компания и сайт;
- список репозиториев.

Если GitHub API временно недоступен или сработал лимит запросов, страница покажет локальный fallback и сообщение вместо списка репозиториев.
