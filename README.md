# kelfy87site

Персональная GitHub-витрина в темной черно-красной эстетике: неоновый фон, 3D-карточка профиля, плавные переходы по разделам, музыкальный плеер с настройкой громкости и живые данные из GitHub API.

## Доступ к сайту

Сайт открывается по ссылке:

<https://kelfy87.github.io/kelfy87site1/>

Проект статический, поэтому для публикации достаточно разместить файлы в GitHub Pages.

## Что внутри

- Живой профиль GitHub: аватар, био, статистика и публичные репозитории пользователя `kelfy87`.
- Верхнее меню с плавными переходами между разделами и активным состоянием текущей секции.
- Топ репозиториев, отсортированный по звездам.
- 3D-tilt карточка профиля.
- Canvas-фон со звездами и мягким красным свечением.
- Музыкальный плеер с кнопкой включения и ползунком громкости.
- Адаптивная верстка для мобильных и десктопных экранов.

## Структура проекта

```text
kelfy87site1/
├── index.html   # Разметка страницы
├── style.css    # Оформление, адаптивность и анимации
├── script.js    # Навигация, музыка, GitHub API и canvas-фон
├── music.mp3    # Локальный аудиотрек
├── skin.png     # Резервный аватар
└── README.md    # Описание проекта
```

## Настройка

Основной профиль задается в начале `script.js`:

```js
const GH_USER = "kelfy87";
```

Чтобы использовать сайт для другого GitHub-профиля, измени значение `GH_USER`.

Локальные ассеты:

- `music.mp3` — трек для музыкального плеера.
- `skin.png` — резервная картинка, если GitHub API недоступен.

## GitHub Pages

1. Открой настройки репозитория.
2. Перейди в раздел **Pages**.
3. Выбери ветку и папку, где лежит `index.html`.
4. Сохрани настройки и дождись публикации.

После публикации сайт будет доступен по ссылке GitHub Pages.

## Технологии

- HTML5
- CSS3
- Vanilla JavaScript
- GitHub REST API
- Canvas API

Проект не требует `npm install`, сборщика или backend-сервера.
